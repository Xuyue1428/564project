Part2 训练及生成
准备环境
python3.6
tensorflow_gpu==1.10.0
cuda9.0+cudnn
pypinyin==0.37.0
tornado==5.0.2
Levenshtein==0.12.0
数据预处理
python prepare_data.py

训练
python train.py -y 五言或七言 -c 藏头或藏尾

例如：python train.py -y 7 -c "TOU"

生成
python generate.py -k 藏头（尾）词 -y 五言或七言 -c 藏头或藏尾

例如：python generate.py -k "江南好" -y "七言" -c "藏头"

界面展示
python start.py
